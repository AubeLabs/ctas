GPKI 사용자용 표준보안API

1. 사용자 인증 및 보안 세션 만들기


  • - 사용자와 서버 인증서 인증 후 보안 세션 만들기 (인증서 로그인)
        사용자 인증서와 서버 인증서에 대한 상호 검증 후 보안 세션을 생성합니다.
       파라미터로 전달된 Form 객체와 URL link 내에 파라미터 값들을
       서버 인증서(공개키)로 암호화합니다.
    • * Form 객체를 이용한 팝업 로그인 (Login)
    • * Form 객체를 이용한 임베디드 로그인 (LoginEmbedded)
    • * URL 링크를 이용한 팝업 로그인 (LoginLink)
