GPKI 사용자용 표준보안API

2. 보안 세션(세션키)을 이용한 암호메시지 생성 및 처리


  • - 클라이언트에서 암호화 메시지 만들어 서버에서 복호화 하기
        보안 세션 구성 후 서버와 클라이언트 간에 교환된 세션키를 이용하여
       Form과 URL link에 포함된 파라미터를 암호화합니다.
       생성된 암호메시지는 gpkiForm에 옮겨져 Target URL 로 전송됩니다.
       전송된 암호메시지는 서버 API에 의해 복호화됩니다.
    • * Form 객체 파라미터 암호화 하기 (Encrypt)
    • * URL link 내 파라미터 암호화 하기 (EncryptLink)
